# km客户端

## 安装依赖

        cd kityminder-editor-master
        npm run init

        cd kmclient
        npm install

## 启动
npm run debug

## 调试
先启动kityminder-editor-master

        cd kityminder-editor-master
        grunt dev

再启动kmclient

        cd kmclient
        npm run debug

## 打包
先将kityminder-editor-master打包

        cd kityminder-editor-master
        grunt build --force
